關於我

我們是一群在台灣的 Ruby 程式語言愛好者，每天的工作都跟 Ruby / Rails 相關。

Ruby on Rails 快速開發的優勢，在國外企業與創業圈極為熱門；而 Ruby 語言本身的自然與優雅簡潔，也讓工程師們工作起來更加快樂。但是，在台灣實際真的有使用 Ruby / Rails 做為主要開發工具的企業卻仍是少數。

這幾年，我們持續在台灣大專院校（成大、輔大、台北商專⋯等校）開授 Ruby 以及 Rails 課程。觀察學生們的學習狀況，大多認為 Ruby 跟 Rails 並不難學，自己看書或照著官網文件練習就能練到一定的程度。但要進階至滿足實戰需求，要在短期間內藉由自學達成並不容易，需要更多的實戰經驗或導師的帶領。因此，在台灣，較資深或有經驗的 Ruby / Rails 開發者相當難找。

近一、二年來，Ruby / Rails 的人才需求增加，年薪超過百萬的高薪職缺紛紛出現，吸引越來越多人渴望進入 Ruby / Rails 的領域。除自行學習以外，更積極參加社群舉辦的聚會與分享、交流。「如果有更快速、更有效的方法能夠成為即戰力就好了！」這樣的需求聲音漸漸出現。於是，幾位愛好 Ruby 的朋友便開始討論「也許我們應該來成立一間專門提供 Ruby / Rails 相關的技術諮詢或是教育訓練的公司」。我們的最終目標，是希望在台灣可以有越來越多企業了解並採用 Ruby 這個優雅、有趣，且極具生產力的程式語言做為主要開發工具，更快速地實現您的想法。
